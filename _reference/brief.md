# Project Brief - Peel Manor House Rebuild

## Client
Owner recently purchased Peel Manor House (Karnup WA). Full written authority given
to Coralstone/Abhishek to use existing site content, photos from client-supplied
drive, and brand info for rebuild.

## Scope
Full rebuild and rebrand of peelmanorhouse.com.au web presence.
NOT hosted publicly during build - local/preview only until client sign-off.

## Inspiration (structure/pattern only, never copy verbatim)
- Marriott Sheraton Grand Mirage weddings page: package tiers, CTA flow, booking structure
- Rafanelli Events: full-bleed photography hero, minimal nav, editorial restraint
- Lily Farm House (Swan Valley venue, WA): rural/timber/gum-tree aesthetic language
  (NOTE: swanvalley.com.au itself is a tourism directory aggregating many venues,
  not a single-venue site - do not reference it directly as a design template)
- Dribbble / Awwwards: landing page animation and interaction patterns

## Photography status
Placeholder phase. Two sources:
1. Client Google/old-computer drive backup - mixed quality, needs sorting
2. Future professional shoot - photographer brief to be generated from placeholder gaps

## Stack
Next.js 15 (App Router) + TypeScript + Tailwind + shadcn
Framer Motion (scroll reveals) + Lenis (smooth scroll)
next/image with blur placeholder, swap-ready for real photos later
Sanity or MDX for Upcoming Events (client self-edit need TBD)

## Order of operations
1. Folder structure (done)
2. Knowledge doc from live site (in progress, 5 pages remaining)
3. Sort client drive photos -> tagged/ by page
4. Flag upscale candidates
5. Pull inspiration references, tag borrowed elements
6. Scaffold Next.js project with placeholders
7. Photographer brief from placeholder gaps
8. Swap real photos in once delivered
